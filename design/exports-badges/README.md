# openkoutsi · PR badges

Personal-record badge assets for the four time-window scopes (all-time, 12 months, 6 months, 3 months) at three tiers (gold, silver, bronze).

## Files

- `pin-<window>-<tier>.svg` — full medal pin, 160×200 viewBox
- `chip-<window>-<tier>.svg` — compact 64×64 chip for inline use
- `*-<size>.png` — raster exports (pin: 400px & 800px wide; chip: 64px & 128px)

Windows: `all-time`, `12mo`, `6mo`, `3mo`
Tiers: `gold`, `silver`, `bronze`

All assets use the openkoutsi accent (#ff5b1f) and a dark base (#0c0c0e). Transparent backgrounds.
